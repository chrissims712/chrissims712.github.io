Trail Updater Help Page

This program works to convert a .gpx file into a feature class with elevation and a hiking difficulty rating calculated for a segment of each part of the trail.

Please note: The program is designed to run on a system which has ArcGIS with the Spatial Analyst extension and Python 2.7 installed.

Instructions:

1. Click the "Input .gpx File" dialog box. 2. Select the appropriate .gpx file for analysis. 3. Click the "Input DEM" dialog box. 4. Select the appropriate DEM file. it is only necessary to select the file with the extension .tif from the example DEM given. 5. Enter into the textbox labeled "Trail Name" the name of the trail being entered into the existing dataset. In this example it is recommended to name the trail "Autumn Ridge." 6. Enter into the textbox labeled "Enter Feature of Existing Trail Dataset" the feature class name of the dataset the trail should be updated. 7. Select the location of the geodatabase the above feature class is located.
